# Path-to-Immortality
网页挂机修仙小游戏 [游戏地址Click Me](https://amovement.github.io/Path-to-Immortality/)

### 介绍
这是一个基于 go 编译的 wasm 浏览器环境下的挂机修仙小游戏（目前也仅支持浏览器运行）, 便于上班摸鱼🐟 ui 选择的基本是暗色调。前端基本全由 Ai 贡献（@豆包），欢迎提交 bug。
数据存储完全依赖本地，无服务端交互，清空本地缓存时会丢失数据，无法找回，请谨慎。

### 目前有的功能
- 境界提升机制（挂机收菜）
- 每日刷新的挑战（日常金币来源）根据境界解锁新的挑战
- 大秘境挑战（金币、锻造武器素材来源）无上限的挑战机制
- 轮回转生机制（每次转生可以达到的角色属性上限更高）鼓励多次转生，提升角色属性上限
- 法器锻造机制（方便拓展角色技能）
